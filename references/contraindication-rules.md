# 治则禁忌规则详解

> 对应知识库 2.2.3 治则与禁忌规则库。
> 单独建库，作为推理引擎的前置拦截层，约束检索范围。所有辨证结果必须先通过本库校验。

## 规则分类（5类，按优先级排序）

### 1. 治则优先级规则（最高层级）

| 场景 | 治则 | 原典依据 |
|:---|:---|:---|
| 表里同病下利清谷 | 急当救里，禁先解表 | 第91条 |
| 表里同病表证为主里证轻 | 先解表后治里 | 第90条 |
| 虚人患病 | 先扶正后祛邪，禁峻攻 | 通用 |
| 急重症 | 优先执行急救治则 | 通用 |

### 2. 绝对禁忌规则（红色拦截，命中直接排除）

> **红/橙/黄三级体系说明**：
> - 🔴 **红色**（原典禁忌）：仲景原文明确禁止，一票否决，直接排除方剂
> - 🟠 **橙色**（后世安全规则）：十八反/十九畏等后世总结，非仲景原典。有原典先例者（如附子+半夏同用于附子粳米汤）降权30%，无先例者降权50%，不直接否决
> - 🟡 **黄色**（相对禁忌）：体质-药性冲突，命中降权+提示

| 禁忌 | 触发条件 | 排除方剂 | 原典依据 |
|:---|:---|:---|:---|
| 咽喉干燥禁汗 | 咽喉干燥 | 麻黄汤类（含麻黄方剂） | 第83条 |
| 淋家禁汗 | 淋家/小便淋沥 | 麻黄汤类（含麻黄方剂） | 第84条 |
| 疮家禁汗 | 疮家/疮疡 | 麻黄汤类（含麻黄方剂） | 第85条 |
| 衄家禁汗 | 衄家/经常鼻衄 | 麻黄汤类（含麻黄方剂） | 第86条 |
| 亡血家禁汗 | 亡血家/失血 | 麻黄汤类（含麻黄方剂） | 第87条 |
| 表证未解禁下 | 恶寒/恶风/脉浮等表证 | 承气汤类 | 通用 |
| 表证未解禁固涩 | 表证未解 | 承气汤类 | 通用 |
| 少阴阳虚禁清热 | 少阴病阳虚里寒 | 白虎汤类 | 通用 |
| 孕妇禁峻下 | 孕妇 | 承气汤类 | 通用 |
| 孕妇禁破血 | 孕妇 | 承气汤类（抵当/桃核承气） | 通用 |
| 孕妇禁剧毒 | 孕妇 | 陷胸汤类（含甘遂等） | 通用 |

**重要**：禁汗规则不仅排除"麻黄汤类"类别，还排除所有组成中含"麻黄"的方剂（如麻黄细辛附子汤、大青龙汤、小青龙汤、葛根汤等）。

### 2.5 后世安全规则（🟠橙色警示，降权而非否决）

> 以下规则源自后世总结（十八反、十九畏），**非仲景原典**。仲景本人有在方剂中同时使用其中部分药对的先例，因此不可一票否决，但应标注警示并降权。

| 互忌药对 | 仲景原典先例 | 处理方式 |
|:---|:---|:---|
| 附子 + 半夏 | ✅ 附子粳米汤（《金匮》） | 降权30%（×0.7），标注"仲景先例" |
| 桂枝 + 赤石脂 | ✅ 乌头赤石脂丸（《金匮》） | 降权30%（×0.7），标注"仲景先例" |
| 乌头 + 半夏 | ❌ 无 | 降权50%（×0.5），无原典先例 |
| 甘草 + 甘遂 | ✅ 甘遂半夏汤（《金匮》） | 降权30%（×0.7），标注"仲景先例" |
| 藜芦 + 细辛 | ❌ 无 | 降权50%（×0.5），无原典先例 |
| 藜芦 + 芍药 | ❌ 无 | 降权50%（×0.5），无原典先例 |
| 硫黄 + 朴硝 | ❌ 无 | 降权50%（×0.5），无原典先例 |
| 人参 + 五灵脂 | ❌ 无（五灵脂非仲景药） | 降权50%（×0.5），无原典先例 |

**完整药对列表及详细规则**见 `assets/rules.json` 中 `category: "postclassical_safety_rule"` 的条目。

**核心原则**：橙色规则不直接否决方剂，但在匹配排名中降权，并在报告中明确标注警示信息。

### 3. 真假寒热鉴别规则（红色拦截，必查项）

详见 `cold-heat-rules.md`。

**核心红色规则**：寒热真假未明时，禁止仅凭表象定寒热治则。

### 4. 相对禁忌规则（黄色警示，命中降权+提示）

|:---|:---|:---|

### 5. 人群禁忌规则

| 人群 | 禁忌 | 级别 |
|:---|:---|:---|
| 老年体虚 | 慎用麻黄等峻猛发汗方剂 | 黄色 |
| 老年体虚 | 慎用大承气等峻猛攻下方剂 | 黄色 |
| 产后经期 | 慎用破血、峻下方剂 | 黄色 |

## 执行顺序

1. 急重症预判（最高优先级，命中即返回）
2. 真假寒热鉴别（必查项，未明则追问）
3. 治则优先级判定
4. 绝对禁忌红色拦截 → 方剂排除清单
5. 后世安全规则🟠橙色警示 → 降权清单（有原典先例×0.7，无先例×0.5）
6. 相对禁忌黄色警示 → 降权清单
7. 人群禁忌适配 → 剂量/风险调整

## 剂量换算标准

采用柯雪帆/郝万山考证：**汉代一两 ≈ 15.625g**

三档换算：
- 常规调理量：原量 × 0.6
- 常规治疗量：原量 × 1.0
- 急证重证量：原量 × 2.0

争议透明：通行教材"一两≈3g"为保守参考，日本汉方"一两≈1g"仅作对照列示。最终剂量由执业医师核定。
