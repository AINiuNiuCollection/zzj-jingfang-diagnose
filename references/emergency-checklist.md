# 急重症判定清单

> 对应知识库 2.2.6 急重症判定清单（v2补强）。
> 第0步"优先排查急重症"须有明确清单逐项对照，命中即优先执行急救治则。

## 判定规则

**命中任一急重症的2个以上判定要点 → 判定为该急重症，优先执行对应急救治则，强制高亮红色警示。**

## 急重症清单

### 1. 少阴三急下证
- **判定要点**：口燥咽干、自利清水色纯青、腹胀不大便
- **原典依据**：第320/321/322条
- **急救治则**：急下存阴
- **主方**：大承气汤

### 2. 蛔厥
- **判定要点**：吐蛔、时烦时止、得食而呕
- **原典依据**：第338条
- **急救治则**：安蛔止痛
- **主方**：乌梅丸

### 3. 蓄血如狂/发狂
- **判定要点**：如狂、发狂、少腹硬满、小便自利
- **原典依据**：第124/125/237条
- **急救治则**：破血逐瘀
- **主方**：抵当汤/桃核承气汤

### 4. 脏结
- **判定要点**：脏结无阳证、舌上白苔滑、难治
- **原典依据**：第129/130条
- **急救治则**：标注难治，提示转诊
- **主方**：无（难治证）

### 5. 亡阳证
- **判定要点**：恶寒脉微而复利、汗出而厥
- **原典依据**：第385条
- **急救治则**：急温回阳救逆
- **主方**：四逆汤辈

### 6. 结胸热实
- **判定要点**：心下痛按之石硬、脉沉紧
- **原典依据**：第135条
- **急救治则**：泻热逐水破结
- **主方**：大陷胸汤

## 执行流程

1. 提取患者核心体征
2. 逐项对照上述清单
3. 命中任一急重症（≥2个判定要点）：
   - 立即优先执行急救治则
   - 红色高亮警示
   - 不进入常规辨证流程
   - 报告中标注急重症预警

## 报告输出格式

```
🚨 急重症预警：[急重症名称]
急救治则：[治则]
建议主方：[方剂]
原典依据：[条文编号]
命中要点：[命中的判定要点]

⚠️ 此为危急重症，请结合临床实际紧急处理！
```
