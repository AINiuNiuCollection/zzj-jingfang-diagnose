# 张仲景审计指南

> 本文档定义"张仲景审计"环节的评审标准与执行细则，对应 SKILL.md 第六步（新增）。审计完成后进入第七步（生成报告）。

## 审计角色定位

以**张仲景**身份进行独立评审——熟悉现代医疗方法，也熟悉运用《伤寒论》《金匮要略》《黄帝内经》《神农本草经》等经典医学著作，强项是结合望闻问切和现代医院检测手段，按古代经典开方剂，对开出的方剂进行审视。

## 核心原则

1. **不刻意迎合**：必须客观、中立地深度分析，不回避问题的核心
2. **整体到具体**：先审视整体治则是否正确，再审视具体方药细节
3. **治人非治病**：评审应先遵循从整体到具体，需分析问题根因由表及里、由此及彼，你治的是人而不是病、你调理的是阴阳而不是头痛医头脚痛医脚
4. **根因溯源**：以经典医学著作为依据、以临床效果为参考，从五行、经典医学著作、临床医案等全方面综合分析
5. **主动追问**：若评审过程中发现信息不足以验证推断，**必须主动询问用户**，待信息完整后再确认结论

## 评审流程

```
推理出的治疗方案和方剂 → 论证患者情况 → 论证结果 → 评审结论 → 评分
```

### 1. 审查治疗方案和方剂
- 审视当前微调后的处方方案
- 检查方剂组成、剂量、煎服法、治疗阶段规划

### 2. 论证患者情况
- 回顾四诊信息与核心病机
- 检查辨证依据是否充分

### 3. 给出论证结果
- 按7个维度逐项打分（见评分标准）
- 每条扣分需附具体依据

### 4. 评审结论
- 通过（≥90分）→ 进入第七步
- 不通过（<90分）→ 回退至第五步半，附改进建议

## 评分标准明细（满分100分）

### ① 整体观（15分）

| 水平 | 得分 | 标准 |
|:---|:---:|:---|
| 优秀 | 13-15 | 从整体到局部，治人非治病、理阴阳非头痛医头；根因分析清晰触及核心病机；治疗方案体现系统性治疗思维 |
| 良好 | 10-12 | 整体观基本到位，根因分析基本合理，偶有局部思维偏移 |
| 需改进 | 7-9 | 存在明显头痛医头倾向，根因分析停留在表面，未触及核心病机 |
| 不合格 | <7 | 无整体观，散点式治疗，未理解"治人非治病"之核心 |

**审视问题**：
- 辨证是从整体出发还是从单一症状出发？
- 方剂是针对病机核心还是针对症状堆砌？
- 是否有"见血止血、见咳止咳"式的对症治疗？
- **多阶段路径是否体现整体观**：是否从患者整体病机层次（标→本→根）出发推演多阶段？阶段之间是否有"先祛邪后扶正""先治标急后治本虚"的整体逻辑？还是每个阶段各自为政、缺乏连贯性？

### ② 原典依据（15分）

| 水平 | 得分 | 标准 |
|:---|:---:|:---|
| 优秀 | 13-15 | 所有结论和药味均可溯源至经典条文；方剂出自原典；加减有原典依据 |
| 良好 | 10-12 | 主方有原典依据，少数辅助信息未标注或引用后世经验 |
| 需改进 | 7-9 | 部分关键结论无原典支撑，或使用后世加减法未标注 |
| 不合格 | <7 | 无原典依据或大量使用后世经验冒充原典 |

**审视问题**：
- 每一味药、每个结论的原典依据是否明确标注？
- 有无无原典凭空添加的药味？
- 剂量换算标准是否标注？
- **多阶段路径的原典依据**：后续阶段推荐的方剂方向是否有原典条文支撑？阶段之间的转方逻辑是否有原典治则条文依据（如SHL-016"随证治之"、SHL-091"急当救里"）？无原典依据的阶段预判须标注"方向性预判，待复诊确认"

### ③ 方证匹配度（15分）

| 水平 | 得分 | 标准 |
|:---|:---:|:---|
| 优秀 | 13-15 | 主方与患者核心病机高度对应；核心指征命中≥3项；排除指征明确不命中 |
| 良好 | 10-12 | 主方匹配良好，核心指征命中≥2项，但非最佳选择（有更优候选） |
| 需改进 | 7-9 | 主方勉强可通，核心指征命中仅1项，或存在排除指征未充分说明 |
| 不合格 | <7 | 方证不匹配或指征明显缺失 |

**审视问题**：
- 主方的核心指征（原文所述的特异性症状）有多少与患者匹配？
- 是否存在比所选方更匹配的候选方？为何不选？
- 排除指征是否已明确排除？

### ④ 药简力专（15分） —— **核心审视：药味是否有核心病机统摄，而非症状堆砌**

> **关键前提**：仲景原方本身药味数差异极大（桂枝汤5味→鳖甲煎丸23味），固定味数红线不成立。评判核心应从"药味是否有病机统摄"出发，而非简单计数。
>
> **仲景原方药味分布**（本知识库280首）：1味62首、2-5味148首、6-9味56首、10-13味10首、14味以上4首（侯氏黑散/麻黄升麻汤14味、薯蓣丸21味、鳖甲煎丸23味）。

**评判规则**：以主方原方药味数为基准（`original_herb_count`），衡量最终处方的药味增量：

| 水平 | 得分 | 标准 |
|:---|:---:|:---|
| 优秀 | 13-15 | 仲景原方直接使用即满分；加减方药味数 ≤ 原方+2味，每味加减均有原典条文依据且针对核心病机变证；药简力专，一药多用 |
| 良好 | 10-12 | 加减方药味数 = 原方+3~4味，多数加减有原典依据，偶有理由不够充分之处；尚可接受但稍显繁复 |
| 需改进 | 7-9 | 加减方药味数 = 原方+5味以上，存在为症状加药、未回归病机核心的情况；有"头痛医头"堆砌嫌疑 |
| 不合格 | <7 | 大量症状对症堆砌无核心病机统摄；或合方推演中药味远超两方合并所需（如两方合并仅需去重复药味，却额外加味5味以上）；违背仲景"方不贵多味"精神 |

**审视问题**：
- **主方原方药味数是多少？最终处方相对原方增加了几味？（增量≤2为佳，≥5需警惕）**
- 若直接使用仲景原方（含大处方如薯蓣丸、鳖甲煎丸），药味数本身不扣分——评判的是加减是否合理，而非原方味数
- 加减药味每味是否有原典条文支撑？是否针对核心病机变证而非对症堆砌？
- 是否存在"见症加药"现象（如：见失眠加酸枣仁、见便秘加大黄、见水肿加防己）而非"见病机加药"？
- 能否用1-2味主药替代当前3-4味同类药？（如：茯苓+猪苓+泽泻+车前子→仅重用茯苓）
- 合方推演时是否正确去重？（如柴胡桂枝汤=小柴胡汤+桂枝汤去重复药味，非简单两方药味相加）

### ⑤ 加减合理性（10分）

| 水平 | 得分 | 标准 |
|:---|:---:|:---|
| 优秀 | 9-10 | 加减均在原典加减法范围内；每项加减有明确原典条文依据；加减理由充分；药味不超过原方+3味 |
| 良好 | 7-8 | 加减基本合理，偶有未标注条文或理由不够充分之处 |
| 需改进 | 4-6 | 存在后世加减法或超出原典范围的加减；药味冗余（超过+5味） |
| 不合格 | <4 | 大量后世加减或无理由增减；违背药简力专原则 |

**审视问题**：
- 加的药味是否在仲景原典加减法中出现过？
- 为何加这味药而不加另一味（如加茯苓而不加猪苓）？
- 是否做到了药简力专？（与④维度交叉验证）

### ⑥ 体质适配（10分）

| 水平 | 得分 | 标准 |
|:---|:---:|:---|
| 优秀 | 9-10 | 充分考虑了BMI/年龄/虚实等体质因素；峻猛药有合理减量或去之方案 |
| 良好 | 7-8 | 基本考虑体质因素，但减量幅度或理由不够充分 |
| 需改进 | 4-6 | 未考虑体质差异，或虚人用猛药 |
| 不合格 | <4 | 完全无视患者体质，按标准剂量开方 |

**审视问题**：
- BMI极低患者是否采用了调理量？
- 峻猛药（大黄、芒硝、麻黄、附子、细辛等）是否做了剂量调整？
- 长期激素史是否影响了辨证判断？

### ⑦ 药性平衡（10分）

| 水平 | 得分 | 标准 |
|:---|:---:|:---|
| 优秀 | 9-10 | 方剂寒热温凉与患者体质协调；寒温并投时比例恰当 |
| 良好 | 7-8 | 药性基本平衡，但有轻微偏颇倾向 |
| 需改进 | 4-6 | 存在明显药性与体质冲突（寒凉方用于脾虚、温燥方用于阴虚） |
| 不合格 | <4 | 药性与体质完全相反 |

**审视问题**：
- 方剂的整体寒热偏性是否与患者的体质匹配？
- 寒热错杂的患者是否采用了寒温并投法？
- 是否有过热或过寒的偏颇？

### ⑧ 治疗方案完整性（10分）

| 水平 | 得分 | 标准 |
|:---|:---:|:---|
| 优秀 | 9-10 | 当前阶段剂量、煎服法、好转标志/变坏标志/转阶段标准、饮食禁忌、调护建议、穴位按摩、运动建议均完整；**多阶段治疗路径完整（含各阶段方剂方向/治则/周期/转方标准），当前所处阶段明确**；**逆向回退路径完整（每阶段有回退标准）；变坏应对路径具体（非仅"停服复诊"）；超期红线明确（每阶段有最大服用期限）；守方vs转方决策清晰**；**西药联用风险评估完整** |
| 良好 | 7-8 | 以上大部分完整，个别项目缺失；或多阶段路径有但不够完整（如缺转方标准）；或逆向回退/变坏应对/超期红线/守方标准中缺1-2项 |
| 需改进 | 4-6 | 部分关键项目缺失（如无预后标志、无煎服法）；或**仅输出当前阶段方案、缺少总体治疗路径和多阶段规划**；或逆向回退/变坏应对/超期红线/守方标准中缺3项以上 |
| 不合格 | <4 | 仅处方无治疗方案；且无多阶段路径；或5项强制质量把控点（逆向回退/西药联用/变坏应对/超期红线/守方决策）均缺失 |

**审视问题**：
- 是否包含明确的当前阶段治疗周期？
- 好转/变坏/转阶段标准是否清晰且可执行？
- 饮食、调护是否针对患者具体体质？
- **是否输出了完整的总体治疗路径（模块0）？各阶段方剂方向、治则、周期、转方标准是否完整？**
- **当前处于第几阶段是否明确标注？**
- **后续阶段的转方标准是否可执行（非模糊表述）？**
- **🔴逆向回退路径：每个阶段是否设计了"若XX则回退至上一阶段"的回退标准？回退标准是否具体可执行？是否引用原典依据？**
- **🔴西药联用风险：是否评估了当前方剂与患者正在服用的西药之间的交互风险？是否列出了需要监测的指标？**
- **🔴变坏应对路径：变坏标志后是否有具体应对方案（而非仅"停服复诊"）？是否包含立即处置+后续方向？**
- **🔴超期红线：每个阶段是否标注了"最大服用期限"？超期后是否明确"必须复诊，禁止自行续服"？**
- **🔴守方vs转方决策：是否设计了守方标准与转方标准的对照？守方标准是否引用原典依据？**

## 打分操作细则

1. **得分必须为整数**（不设小数），每项独立评分后求和
2. **扣分须注明具体依据**（如"原典依据扣5分：加味XXX无原典条文支撑"）
3. **首次审计自动标记为"张仲景审计·第1轮"**
4. **不通过时须给出明确改进建议**，指出回退到第五步半后应修改哪些方面
5. **第3轮仍未通过时**，输出全部3轮审计历史（含评分变化与改进记录），标注"建议执业医师人工裁断"
6. **通过后**，将评分结果纳入报告模块8（可追溯元信息）

## 常见否决场景

| 场景 | 说明 | 扣分方向 |
|:---|:---|:---|
| 方证不匹配 | Top1方核心指征命中<2项 | ③方证匹配度直接不及格 |
| 无原典加减 | 添加的药味无原典依据（如加当归、川芎） | ⑤加减合理性≤4分 |
| 加减冗杂 | 加减药味≥原方+5味且无核心病机统摄，或合方未去重致药味膨胀 | ④药简力专≤6分 |
| 虚人用峻剂 | BMI<17未减量，或直接给标准治疗量 | ⑥体质适配≤4分 |
| 纯寒纯温偏颇 | 完全无视患者寒热错杂体质 | ⑦药性平衡≤6分 |
| 无总体治疗路径 | 仅输出当前阶段方案，缺少模块0总体治疗方案（多阶段路径/当前所处阶段/转方标准） | ⑧治疗方案完整性≤6分 |
| 多阶段路径无原典依据 | 后续阶段方剂方向无原典条文支撑，或转方逻辑无原典治则条文 | ②原典依据≤10分 |
| 无预后标志 | 仅处方无好转/变坏/转阶段标准 | ⑧治疗方案完整性≤4分 |
| 无逆向回退路径 | 多阶段路径中未设计"若XX则回退至上一阶段"的回退标准 | ⑧治疗方案完整性≤6分 |
| 无变坏应对路径 | 变坏标志后仅写"停服复诊"，无具体应对方案（立即处置+后续方向） | ⑧治疗方案完整性≤6分 |
| 无超期红线 | 阶段未标注最大服用期限，或未明确"逾期必须复诊" | ⑧治疗方案完整性≤6分 |
| 无守方决策 | 仅有转方标准而无守方标准，患者可能急于转方而错失守方再效之机 | ⑧治疗方案完整性≤7分 |
| 无西药联用风险评估 | 患者正在服用西药（如激素）但报告中无联用风险评估 | ⑧治疗方案完整性≤6分 |
| 头痛医头 | 方剂为症状堆砌，无核心病机思路 | ①整体观≤10分 |
