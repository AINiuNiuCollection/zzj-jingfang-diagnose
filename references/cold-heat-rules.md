# 真假寒热鉴别规则

> 对应知识库 2.2.3 真假寒热鉴别规则（v2补强，红色拦截，必查项）。
> 依据原典第11条，杜绝阴盛格阳/阳盛格阴导致的致命性治则颠倒。

## 原典依据

**第11条**："病人身大热，反欲得近衣者，热在皮肤，寒在骨髓也；身大寒，反不欲近衣者，寒在皮肤，热在骨髓也。"

## 鉴别要点（四维交叉验证）

### 1. 寒热表象与欲衣情况
- **身大热 + 反欲近衣** → 真寒假热（热在皮肤，寒在骨髓）
- **身大寒 + 反不欲近衣** → 真热假寒（寒在皮肤，热在骨髓）
- 寒热与欲衣一致 → 可能为真寒/真热

### 2. 口渴与饮水情况
- **真热**：口渴喜冷饮
- **真寒假热**：口渴不欲饮，或欲饮热
- **真热假寒**：口渴喜冷饮

### 3. 脉象是否与寒热表象一致
- **真寒假热**：脉沉微、脉沉迟（虽身大热而脉无力）
- **真热假寒**：脉滑数有力、脉沉数（虽身大寒而脉有力）
- 脉象与寒热表象一致 → 支持真寒/真热

### 4. 小便与舌象
- **真寒**：小便清长、舌淡苔白滑
- **真热**：小便短赤、舌红苔黄燥

## 红色规则

**寒热真假未明时，禁止仅凭表象定寒热治则，触发智能追问至鉴别信息充足。**

## 触发条件

### 需要触发寒热鉴别的情况
1. 出现极端寒热表现：身大热、高热、身大寒、四肢厥冷
2. 寒热表现与欲衣情况矛盾
3. 脉象与寒热表象矛盾

### 不需要触发的情况
- 普通外感（发热恶寒但非极端）
- 无寒热表现
- 寒热表现明确且脉象/渴饮一致佐证

## 追问策略

当寒热真假未明时，优先追问最能消歧的鉴别点：

1. **欲衣情况**（最关键）："患者虽发热，是否反而想多穿衣服？"
2. **脉象**："脉象是沉微无力还是洪大有力？"
3. **饮水情况**："口渴吗？想喝冷的还是热的？"
4. **小便情况**："小便颜色如何？清长还是短赤？"

## 鉴别结论分类

| 结论 | 含义 | 后续治则 |
|:---|:---|:---|
| 真热 | 热证，可清热 | 白虎汤类、承气汤类 |
| 真寒 | 寒证，可温阳 | 四逆汤类、理中丸类 |
| 真寒假热 | 本质是寒，表象有热 | 温阳散寒，禁清热 |
| 真热假寒 | 本质是热，表象有寒 | 清热泻火，禁温阳 |
| 寒热真假未明 | 信息不足 | 追问，不定寒热治则 |

## 注意事项

1. 舌诊在仲景原典中为辅助信息（原典舌诊记载零散），舌象判断需标注来源层级
2. 脉证合参，不做简单加权平均，亦不套用后世"舍证从脉/舍脉从证"
3. 脉证不一致时触发多假设追踪，不机械舍一
