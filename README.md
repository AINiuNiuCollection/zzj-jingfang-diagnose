# zzj-jingfang-diagnose

基于宋版《伤寒论》《金匮要略》原典的**张仲景经方辨证辅助系统**，为执业中医师提供临床辨证参考。

> **定位红线**：辅助参考工具，非独立诊疗系统。最终诊疗决策权归属执业医师。所有输出标注"仅供临床参考"。

## 功能概览

| 功能 | 说明 |
|:---|:---|
| **经方辨证** | 用户提供四诊信息，按八步流程完成从治则判定到处方输出的全过程 |
| **条文查询** | 按编号/方剂/症状/关键词/篇章查询原典条文 |
| **类方鉴别** | 对比相似方剂的核心指征、排除指征、组成、病机等维度 |

## 核心流程：八步辨证

| 步骤 | 内容 | 关键点 |
|:---|:---|:---|
| 1 | 治则优先 + 急重症预判 + 真假寒热鉴别 | 急重症一票优先；寒热未明不定治则（第11条） |
| 2 | 信息标准化（术语映射） | 白话→文言标准化，分五类权重（主症/兼症/脉象/舌象/否定症状） |
| 3 | 抓主症，定六经病位 | 多假设追踪，置信度<0.65触发追问；合病分主次 |
| 4 | 误治史适配校验 | 汗后/下后/火逆等变证匹配 |
| 5 | 方证精准匹配 | Top3候选方，核心指征一票定方，验证性追问 |
| 6 | 禁忌二次复核 | 红黄蓝三级风险标注，药性适配检查 |
| 7 | 经方合理微调 | 体质调量 + 原典加减法推演 + 合方推演（≤13味药） |
| 8 | **张仲景审计** | 8维度100分评审，≥90分通过，否则回退重推，最多3轮 |

### 张仲景审计评分维度（满分100分）

| 维度 | 满分 | 审视要点 |
|:---|:---:|:---|
| 整体观 | 15 | 治人非治病、理阴阳非头痛医头；根因分析触及核心病机 |
| 原典依据 | 15 | 所有结论和药味可溯源至经典条文 |
| 方证匹配度 | 15 | 主方与核心病机高度对应，核心指征命中率 |
| 药简力专 | 15 | 方剂≤13味；超13味扣分，超19味不合格 |
| 加减合理性 | 10 | 加减在原典加减法范围内，每项有原典条文依据 |
| 体质适配 | 10 | 剂量考虑BMI/年龄/虚实；峻猛药减量或去之 |
| 药性平衡 | 10 | 寒热温凉与患者体质协调；无纯寒纯温偏颇 |
| 治疗方案完整性 | 10 | 剂量/煎服法/好转变坏标志/转阶段标准/调护建议完整 |

## 使用方式

### 经方辨证

提供患者四诊信息，系统按八步流程生成论证式辨证报告：

- **必填核心项**：寒热、汗出、脉象、二便
- **选填拓展项**：舌象、欲近衣情况、渴饮、人群属性、治疗史、既往病史

### 条文查询

```bash
# 按条文编号
python scripts/query_clause.py --clause-id "SHL-012"

# 按方剂名
python scripts/query_clause.py --formula "桂枝汤"

# 按症状
python scripts/query_clause.py --symptom "往来寒热"

# 按关键词全文搜索
python scripts/query_clause.py --keyword "心下痞"

# 按篇章
python scripts/query_clause.py --chapter "辨太阳病"
```

### 知识库检索（AI内部调用）

```bash
python scripts/retrieve_kb.py \
  --chief "主诉" \
  --symptoms "症状1,症状2" \
  --pulse "脉象" \
  --chill-fever "寒热" \
  --sweat "汗出" \
  --stool-urine "二便" \
  --want-clothing "欲近衣" \
  --thirst "渴饮" \
  --population "人群" \
  --treatment-history "治疗史" \
  --extra "其他补充"
```

输出 JSON 包含：急重症判定、真假寒热鉴别、禁忌排除、六经候选、Top5候选方剂、Top10召回条文、验证性追问、信息完整性检查、患者体质适配分析。

## 剂量换算标准

采用柯雪帆/郝万山考证：**汉代一两 ≈ 15.625g**

| 用量级别 | 换算系数 | 适用场景 |
|:---|:---|:---|
| 常规调理量 | 原量 × 0.6 | 体质虚弱、老年患者 |
| 常规治疗量 | 原量 × 1.0 | 一般成人 |
| 急证重证量 | 原量 × 2.0 | 急证重证 |

争议透明：通行教材"一两≈3g"为保守参考，日本汉方"一两≈1g"仅作对照。最终剂量由执业医师核定。

## 红线约束

1. **唯一诊疗依据**：宋版原典，不混入后世医家经验
2. **禁忌100%拦截**：一票否决，无方证用反、治则颠倒
3. **寒热必查**：第11条真假寒热未明不定治则
4. **100%可溯源**：无条文支撑的内容禁止输出
5. **信息不全不强行结论**：触发智能追问（六经追问 + 验证性追问双机制）
6. **脉证合参**：非后世"舍证从脉"，脉证冲突触发多假设追踪
7. **舌诊降为二级辅助**：原典舌诊稀疏，不泛化舌象-方剂关联
8. **整体观原则**：治人非治病、理阴阳而非头痛医头
9. **药简力专**：方剂≤13味，超则为未抓核心病机
10. **微调须原典依据**：不得引入后世加减法，须标注对应条文编号
11. **既往病史仅辅助参考**：不作为独立辨证依据，不改变原典溯源承诺
12. **张仲景审计强制执行**：微调后必须审计（≥90分通过），不通过回退重推

## 已知局限

1. 仅擅长典型经方证匹配，症状极不典型时匹配度下降
2. 方剂微调仅限于仲景原典框架内的条文推演，不包括后世医家加减经验
3. 不包含后世医家经验，仅代表仲景原典体系
4. 舌诊等原典稀疏维度，部分判断需参考后世补充（已标注层级）
5. 仅覆盖高频误治变证，复杂坏病暂不支持
6. 剂量换算存在学界争议，最终剂量须执业医师核定
7. 药性适配分析仅基于原典方剂组成药味，不引入后世药性归经理论扩展
8. 体质适配分析标注"辅助参考"层级，不作为独立辨证依据
